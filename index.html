This check-in system now runs as a server

To support multiple check-in stations and live Google Sheet sync, this project moved from a single static page to a small local server.

Start it with:

cd server && python3 server.py

Then open the address it prints (e.g. http://localhost:5050) — or, from another device on the same WiFi, the network address it prints.

See SETUP.md in this project for full instructions, including connecting a Google Sheet.
